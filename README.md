# Ravi Abbineni Website

This site is a static GitHub Pages-friendly homepage.

## Content vs Presentation
- Content files live in `content/`.
- Presentation files live in `assets/css/styles.css` and `assets/js/site.js`.
- `index.html` contains page structure, mount points, and SEO metadata only.

## Updating Content
- Profile copy: edit `content/profile.json`
- Focus cards: edit `content/focus.json`
- Perspective entries: edit `content/perspective.json`
- Activity entries: edit `content/activity.json`

## Runtime
- Supported in GitHub Pages
- Supported in any simple local static server
- Not designed for direct `file://` browser opening
